Hergebruik via open standaarden

Alle gepubliceerde jottems en annotaties zijn machineleesbaar beschikbaar via IIIF, RSS, Elasticsearch, W3C Web Annotations en schema.org (AP NDE). Basis-URL https://api.iotm.nl/v1.

IIIF Change Discovery RSS-feed Elasticsearch zoeken W3C Web Annotations IIIF Collection / Manifest Datasetbeschrijving

IIIF Change Discovery

Periodiek nieuwe/gewijzigde jottems en annotaties harvesten.

GET /organisatie/smh/jottems/iiif/change-discovery 
// Activity Stream (OrderedCollection)
{ "type": "OrderedCollectionPage",
  "orderedItems": [
    { "type":"Update", "object":{ "id":"https://ark.iotm.nl/ark:/27023/g8x2" } }
  ] }

RSS-feed

Nieuwe jottems per organisatie volgen.

GET /organisatie/smh/jottem/rss 
<item>
  <title>Chinees-Indisch restaurant Lotus</title>
  <link>https://ark.iotm.nl/ark:/27023/g8x2</link>
</item>

Elasticsearch zoeken

POST /jottem/search 
{ "query": { "match": { "name": "shoarma" } } }

W3C Web Annotations

Annotaties ophalen via het W3C Web Annotation Protocol.

GET /annotation/{id} 
{ "@context":"http://www.w3.org/ns/anno.jsonld",
  "type":"Annotation",
  "motivation":"commenting",
  "body":{ "value":"Mijn vader begon dit restaurant in 1973." },
  "target":"https://ark.iotm.nl/ark:/27023/g8x2" }

IIIF Collection / Manifest

GET /organisatie/smh/jottems/iiif/collection
GET /jottem/g8x2/iiif/manifest 
{ "@context":"http://iiif.io/api/presentation/3/context.json",
  "type":"Manifest", "label":{ "nl":["Restaurant Lotus"] } }

Datasetbeschrijving (schema.org AP NDE)

Volledige datadump als RDF, harvestbaar door aggregators (NDE).

GET /organisatie/smh/dataset 
{ "@context":"https://schema.org/", "@type":"Dataset",
  "name":"Smaak van Gouda",
  "distribution":[{ "encodingFormat":"application/n-triples" }] }

De volledige OpenAPI-specificatie staat in het ontwerpdocument (openapi.yaml · Swagger UI).
